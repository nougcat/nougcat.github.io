# Dowód $e^x$


$$\lim_{x \to \infty}(1+\frac{c}{n})^n$$
załóżmy, że k = $\frac{n}{c}$

$$\lim_{n \to \infty} (1 + \frac{1}{k})^{kc} = (\lim_{k \to \infty} (1 + \frac{1}{k})^{k})^c = e^c$$
